# HADES-SA_SpinnyONE
Source code and Linux/Windows binaries for the HADES-SA (SpinnyONE) satellite decoding software

The program accepts as a parameter a file containing an already descrambled and CRC-checked byte sequence received by an FSK demodulator compatible with the satellite transmissions. Any FSK software can be used, but this program is intended to work with Andy's UZ7HO Soundmodem software for Windows.
Some sample TXT packet files are included. The Linux and Windows (.exe) binaries are provided together with the source code.
For any questions, please contact us at contacto@amsat-ea.org
